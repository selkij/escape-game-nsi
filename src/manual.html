NMAP

🔎 Scan de base

nmap cible # Scan les ports ouverts sur une machine

🎯 Types de scans

nmap -sS cible # Scan furtif SYN (par défaut en root)
nmap -sT cible # Scan en mode connect() (moins furtif)
nmap -sU cible # Scan UDP
nmap -sV cible # Détection des versions des services
nmap -A cible # Scan avancé (OS + version + traceroute + scripts)
nmap -O cible> # Détection du système d’exploitation

⚡ Optimisation et furtivité

nmap -T4 cible # Scan rapide
nmap -T2 cible # Scan lent (furtif)
nmap --min-rate 500 cible # Accélération en fixant un taux minimum de paquets/s
nmap --max-retries 1 cible # Réduire le nombre de tentatives par port

🛠 Utilisation de scripts NSE

nmap --script=vuln cible # Recherche de vulnérabilités connues
nmap --script=http-enum cible # Enumération des services web
nmap --script smb-os-discovery cible # Infos sur le SMB

📜 Sortie et rapports

nmap -oN scan.txt cible # Sauvegarde en texte brut
nmap -oX scan.xml cible # Sauvegarde en XML
nmap -oG scan.gnmap cible # Format greppable

🏠 Scan d’un réseau entier

nmap 192.168.1.0/24 # Scanner tout un sous-réseau
nmap -sn 192.168.1.0/24 # Ping scan (trouve les machines actives)
